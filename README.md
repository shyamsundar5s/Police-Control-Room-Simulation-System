# Police Control Room Simulation 🚔

A Python-based simulation of an emergency police dispatch system that handles incident reporting, priority classification, and unit allocation.

## Features
- Incident classification using AI logic
- Real-time dispatch simulation
- Police unit management
- Incident tracking dashboard

## Tech Stack
- Python
- OOP
- Basic AI (rule-based)

## Future Scope
- ML-based prediction
- Real-time tracking
- Web dashboard
